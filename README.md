Descripción del proyecto

Este trabajo práctico consiste en desarrollar una aplicación de consola en C# que analiza datos de Fórmula 1 utilizando un archivo CSV con resultados reales de los últimos años (2016–2024).

El programa permite cargar el dataset completo y realizar distintas consultas como:

- Podios logrados por un piloto  
- Puntos obtenidos por un equipo en un año específico  
- Mejor remontada del dataset  
- Lista ordenada de equipos participantes  
- Visualización completa del dataset  
- Promedio de mejora por piloto (funcionalidad del coloquio)

El objetivo final es aplicar estructuras de datos, manejo de archivos, ciclos, condicionales y operaciones con matrices.

-----------------------------------------------------------------------------------------------------------------------

Funcionalidad del Coloquio: Promedio de mejora por piloto

Se agregó una nueva opción en el menú:

7. Promedio de mejora por piloto

Esta funcionalidad calcula:

- La cantidad de posiciones ganadas o perdidas por un piloto en cada carrera  
- El promedio de mejora (o empeoramiento) por carrera  
- La cantidad total de carreras consideradas  

Interpretación del resultado:

- Si el número es positivo = ganó posiciones
- Si el número es negativo = perdió posiciones 
- Si el número es 0 = mantuvo su posición promedio
